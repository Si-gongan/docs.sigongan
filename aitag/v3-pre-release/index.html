1. 개요

안녕하세요, aitag 개발팀입니다.

aitag 대체텍스트 생성 서비스가 v3.0.0으로 업그레이드됩니다. 이번 버전은 새로운 AWS 인프라 기반으로 재구축되어, 안정성, 확장성, 운영 효율성이 크게 향상됩니다.

또한 웹 기반 사용자 인터페이스가 새롭게 추가되어, 더욱 편리하게 서비스를 이용하실 수 있습니다.

2. 주요 변경사항

2.1 기존 vs 신규 비교

항목 기존 aitag v3.0.0
인프라 EC2 + Docker 새 AWS 인프라 (EC2, VPC, S3, Redis)
인증 없음 (외부 API 인증) JWT + API Key
관리 도구 없음 Admin UI
이력 관리 파일 로깅만 히스토리 UI
대량 처리 내부용만 일괄처리 UI
클라이언트 API만 제공 Streamlit 웹 UI
모니터링 Slack 알림만 Sentry + Slack 알림

참고: AI 모델은 기존과 동일하게 외부 LLM API(Gemini)를 사용합니다.

3. 신규 기능 상세

3.1 웹 기반 사용자 인터페이스

기존 API 방식에서 벗어나, 웹 브라우저에서 직접 서비스를 이용할 수 있습니다.

기능 설명
로그인/인증 개인 계정으로 로그인하여 사용 이력 관리
메인 서비스 이미지 업로드 후 대체텍스트 즉시 생성
히스토리 이전 생성 결과 검색 및 재사용
대량 업로드 여러 이미지를 한 번에 처리
Admin 사용자 및 클라이언트 관리 (관리자 전용)

3.3 히스토리 기능

모든 대체텍스트 생성 결과가 저장되어, 언제든지 조회하고 재사용할 수 있습니다.

주요 기능

  • 날짜 범위, 키워드, AI 모델별 검색
  • 이전 결과 상세 보기 및 복사
  • 생성 이력 통계 확인

기대 효과

  • 동일 이미지 재처리 불필요
  • 과거 결과 참조 및 일관성 유지
  • 사용량 현황 파악

4. 사용 가이드 (예정)

서비스 URL과 계정 정보는 런칭 전 별도 안내드립니다.

지원 이미지 형식: JPG, PNG, GIF, WebP

4.1 로그인

  1. 웹 브라우저에서 aitag 서비스 URL 접속
  2. 발급받은 계정 정보로 로그인
  3. 로그인 성공 시 메인 서비스 화면으로 이동

4.2 대체텍스트 생성

  1. 메인 화면에서 이미지 업로드 영역 클릭 또는 드래그 앤 드롭
  2. AI 모델 확인 (Gemini)
  3. 생성 버튼 클릭
  4. 결과 확인 후 복사 또는 다운로드

4.3 히스토리 조회

  1. 상단 메뉴에서 '히스토리' 클릭
  2. 검색 조건 설정 (날짜, 키워드, 모델 등)
  3. 검색 결과에서 원하는 항목 클릭
  4. 상세 보기에서 결과 복사 또는 재생성

4.4 대량 업로드

  1. 메인 화면에서 '대량 업로드' 탭 선택
  2. 여러 이미지 파일 선택
  3. AI 모델 선택 후 '일괄 생성' 클릭
  4. 진행률 확인 (처리 중 다른 작업 가능)
  5. 완료 알림 후 결과 다운로드

5.3 Week별 상세

Week 1: 기반 구축 (1/12 - 1/18)

  • AWS 인프라 구축 (EC2, VPC, 보안그룹)
  • S3, Redis 설정
  • JWT + API Key + RBAC 인증 시스템
  • 로그인 화면 및 세션 관리

Week 2: 핵심 기능 (1/19 - 1/25)

  • Admin API 및 UI (사용자/클라이언트 관리)
  • 메인 서비스 UI (이미지 업로드 → 대체텍스트 생성)
  • 히스토리 API 및 UI (검색, 필터, 조회)

Week 3: 완성 (1/26 - 2/1)

  • 일괄처리 API 및 대량 업로드 UI
  • Sentry 에러 모니터링 + Slack 알림 연동
  • 성능 최적화

Week 4: QA 및 배포 (2/2 - 2/10)

  • 전체 기능 통합 테스트
  • 버그 수정 및 안정화
  • 스테이징 환경 배포 및 검증
  • 운영 환경 배포 준비

참고: 상기 일정은 예정이며, 변경 시 사전 안내드리겠습니다.
