개요

로그인 후 가장 먼저 보이는 메인 화면.

화면 구조

영역내용스크롤
검색 영역검색창 + 마이크-
최근 본 상품최근 조회 상품 목록가로
추천 상품AI 추천 상품세로 (메인)
오늘의 특가할인 상품가로
하단 탭 바네비게이션고정

검색 영역

UI 구성

요소내용
검색 입력창Placeholder: "상품을 검색해보세요"
마이크 아이콘음성 검색 (우측)

동작

액션결과
검색창 터치검색 화면으로 이동
마이크 터치음성 검색 시작

접근성

항목처리
검색창"검색창, 버튼. 상품을 검색해보세요. 이중 탭 하여 검색 화면 열기"
마이크"음성으로 검색, 버튼"

최근 본 상품

노출 조건

  • 상품 상세를 1회 이상 조회한 경우
  • 조회 이력이 없으면 섹션 미노출

UI 구성

요소내용
섹션 타이틀"최근 본 상품" (Heading)
상품 카드가로 스크롤 리스트
최대 개수10개

상품 카드 (소형)

요소내용
썸네일정사각형 이미지
상품명최대 2줄, 말줄임
가격판매가

동작

액션결과
상품 카드 터치상품 상세 이동
가로 스와이프목록 스크롤

접근성

항목처리
섹션"최근 본 상품, 가로 목록. 스와이프 하여 더 보기"
상품"상품명, 가격원. 이중 탭 하여 상세 보기"

추천 상품

UI 구성

요소내용
섹션 타이틀"추천 상품" (Heading)
상품 카드2열 그리드
로딩무한 스크롤

상품 카드 (그리드형)

요소내용
썸네일정사각형 이미지
상품명최대 2줄, 말줄임
가격판매가

무한 스크롤

상태표시
로딩 중하단 스피너
데이터 없음"더 이상 추천 상품이 없습니다"

오늘의 특가

UI 구성

요소내용
섹션 타이틀"오늘의 특가" (Heading)
상품 카드가로 스크롤 리스트

상품 카드 (특가형)

요소내용
썸네일정사각형 이미지
할인 뱃지"N% 할인" (빨간색)
상품명최대 2줄, 말줄임
원가취소선
판매가강조 표시

접근성

항목처리
상품"상품명, N% 할인, 원가 N원에서 판매가 N원"

상품 카드 공통

터치 영역

  • 전체 카드 영역 터치 가능
  • 최소 터치 영역: 44x44pt

이미지 로딩

상태표시
로딩 중회색 placeholder
로딩 실패기본 이미지 아이콘

핀치 줌 기능

목적

범위
